# Keith
A CSS Library inspired by duck-related colours.

Made with HTML, CSS, JS and Prism.js.

## Colour Palette
![Colour palette](image.png)

- Duck: #FEDE00
- Grass: #0A7029
- Swamp Water: #DBE8D8
- Duckweed: #C8DF52

## Current Components
- Buttons
- Percentage bars
- Modals
